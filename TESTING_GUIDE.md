## 🐛 常见问题

### Q: 详情页一直显示"加载中"
**A**: 检查 API 是否正常，打开浏览器控制台查看错误信息

### Q: 点击文章跳转到了外部网站
**A**: 代码未正确更新，检查是否所有文件都已保存

### Q: 详情页显示"文章未找到"
**A**: 可能是 ID 不匹配，检查 URL 中的 ID 是否在数据列表中

### Q: 封面图不显示
**A**: 可能是图片加载失败，检查网络或图片 URL
