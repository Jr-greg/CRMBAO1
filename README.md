# 入门宝 - 加密货币新手入门指南

一个完整的静态 UI 原型项目，使用 Next.js 14 + TypeScript + TailwindCSS 构建。

## 快速开始

```bash
# 安装依赖
npm install
# 或使用 pnpm
pnpm install

# 启动开发服务器
npm run dev
# 或
pnpm dev
```

打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 技术栈

- **Next.js 14** (App Router)
- **TypeScript**
- **TailwindCSS**
- **Lucide React** (图标库)

## 项目说明

这是一个**纯静态 UI 原型**，包含：
- ✅ 所有页面的完整 UI
- ✅ 响应式设计（移动端/桌面端）
- ✅ 深色/浅色主题切换
- ✅ shadcn/ui 风格的组件库
- ❌ 不包含真实数据逻辑
- ❌ 不包含后端接口
- ❌ 按钮点击仅做 UI 演示

## 页面路由

- `/` - 首页
- `/learn` - 新手入门中心
- `/binance` - Binance 教程
- `/okx` - OKX 教程
- `/tools` - 实用工具
- `/faq` - 常见问题
- `/disclosure` - 合规披露
